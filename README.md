## MP3 Player
Play local MP3 files on Android devices. APK for the latest release can be found [here](https://github.com/Serious-senpai/mp3-player/releases/latest).
### Supported features
- Play local MP3 audio files in background
- Create playlists from MP3 files
- Repeat and shuffle mode
- Download audio from YouTube videos to device
  - Play YouTube music in background without Premium even when offline
